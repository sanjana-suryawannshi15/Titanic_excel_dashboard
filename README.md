# Titanic_excel_dashboard

📘 Project Overvie
This project is an interactive Excel dashboard based on the famous Titanic dataset. It visualizes key insights into passenger demographics and survival patterns using Excel tools such as PivotTables, PivotCharts, Slicers, and Conditional Formatting. The goal of this project is to demonstrate data cleaning, analysis, and visualization skills using Microsoft Excel.

**🎯 Objectives**
 * Analyze survival trends among Titanic passengers.
 * Identify how factors like gender, passenger class, and age affected survival chances.
 * Create a clear and interactive Excel dashboard for data-driven storytelling.

**📂 Dataset Information**
Dataset: Titanic Passenger Dataset
Source: Kaggle (Titanic: Machine Learning from Disaster)
Total Records: 891 passengers
Main Columns:
  * PassengerId – Unique ID for each passenger
  * Survived – Survival status (0 = No, 1 = Yes)
  * Pclass – Passenger class (1st, 2nd, 3rd)
  * Name, Sex, Age, SibSp, Parch, Fare, Embarked

**🧹 Step 1: Data Cleaning**
Performed in Excel:
 1. Removed duplicate rows and blank entries.
 2. Replaced missing values in Age and Embarked with averages/modes.
 3. Converted categorical values (like “Survived” and “Pclass”) into readable labels.
 4. Ensured data types were consistent for numerical and text columns.


**📊 Step 2: Data Analysis**
 * Key metrics calculated using Excel formulas and PivotTables:
 * Average Age of Survivors vs Non-Survivors.
 * Count of passengers by Class, Gender, and Embarkation Port.
 * Survival Rate (%) by Gender, Class, and Age Group.

**🧠 Step 3: Dashboard Creation**
 Built an interactive dashboard with the following visualizations:
1. Survival Rate by Gender – Bar chart
2. Passenger Distribution by Class – Pie chart
3. Age vs Survival – Scatter or histogram chart
4. Embarkation Port Analysis – Column chart
5. Added Slicers for filtering by:
  * Gender
  * Class
  * Embarked Port

**📈 Step 4: Insights & Findings**
   🎯 Women had a much higher survival rate than men.
   💎 1st Class passengers were most likely to survive.
   👶 Younger passengers had slightly higher survival chances.
   ⚓ Most passengers embarked from Southampton (S).
  

**⚙️ Step 5: Tools & Techniques Used**
   1. Microsoft Excel 2013+
   2. Pivot Tables and Pivot Charts
   3. Data Cleaning (Remove Duplicates, IF, AVERAGEIF, COUNTIFS)
   4. Slicers and Filters
   5. Dashboard Design and Formatting

**💡 Project Outcomes**
* Built a clean and interactive dashboard for quick exploration.
* Practiced Excel-based data analysis and visualization skills.
* Gained insights into how demographics affected Titanic survival rates.
